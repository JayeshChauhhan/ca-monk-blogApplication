# CA Monk Blog Application

A premium, modern blog application showcasing full-stack development skills.

## Features
- Real-time blog list with beautiful card design
- Blog detail view with full content
- Create new articles with modal form
- TanStack Query for efficient data fetching
- TypeScript for type safety
- Tailwind CSS for premium styling

## Tech Stack
- Next.js 16
- React 19
- TypeScript
- TanStack Query
- Tailwind CSS
- JSON Server (mock backend)

## Setup
npm install
npm run dev
npx json-server --watch db.json --port 3001
